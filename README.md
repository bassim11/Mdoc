## 🚀 النشر على Vercel

### الطريقة الأولى: من GitHub
1. ارفع المشروع على GitHub
2. اذهب إلى [vercel.com](https://vercel.com) وسجّل دخول
3. اضغط **New Project** واختر المستودع
4. اضغط **Deploy** — سيتم النشر تلقائياً

### الطريقة الثانية: Vercel CLI
```bash
npm i -g vercel
cd mdoc-project
vercel
```

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| 📋 ملف الموظف الكامل | البيانات الشخصية والوظيفية والرواتب |
| 🎯 متابعة الترقية | حساب تلقائي لتاريخ الاستحقاق |
| 📚 الدورات التدريبية | نوع الدورة، المحاضر، الأيام، الساعات |
| 📁 الملفات الرسمية | كتب، شكر، أوامر، شهادات |
| 🪪 الملفات الشخصية | هوية الدائرة، بطاقة وطنية، سكن، راتب |
| 🔍 بحث سريع | بالاسم أو الرقم الوظيفي |
| 💾 حفظ دائم | localStorage — لا تُحذف البيانات |
| ☀️🔄 نوع الدوام | موظف صباحي أو مناوب |

## 🛠️ التقنيات المستخدمة

- **HTML5** — بنية الصفحة
- **CSS3** — التصميم والأنماط
- **Vanilla JavaScript** — جميع الوظائف
